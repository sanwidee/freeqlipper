# Qlipper AI v1.8.5

AI-powered video clipping tool for content creators.

## Quick Start

### macOS
1. Double-click `run-mac.sh` or `Qlipper-AI.command`
2. If blocked, right-click → Open
3. Open browser to http://localhost:3001

### Windows
1. Double-click `run-windows.bat`
2. Open browser to http://localhost:3001

## Migrating from Previous Version

To keep your downloaded videos and transcripts:
1. Copy the `assets/` folder from your old installation
2. Paste it into this new folder (replace the empty one)

## Requirements
- Node.js 18+ (will auto-install dependencies)
- FFmpeg (auto-installed via npm on first run)
- yt-dlp (auto-downloaded on first run)
